Chrome API vsdoc .vsdoc file for Chrome API

This project was designed to aid authors in the creation of Google Chrome Extensions when authoring in the Visual Studio IDE. By referencing the chrome-api-vsdoc.js file in your extension's source code, Visual Studio can provide intellisense when programming against the Google Chrome APIs.

Games

Connected Worlds HTML5 2D Platformer for Ludum Dare

This is a simple 2D platformer completed for the Ludum Dare 48 hour competition. The theme was Connected Worlds and you could interpret that how you wished. The idea behind this game is that Switch, the hero, can teleport inbetween parallel worlds at will. The two worlds are similar, but not identical. Switch has to use this ability to progress through each level by solving puzzles. This was my offical #LD30 entry.

The code, art and sound were created in roughly 20 hours over the course of 2 days.
